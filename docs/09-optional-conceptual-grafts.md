# Tier 3.1 — Optional Additional Conceptual Grafts

Low priority follow-up ideas after the current conceptual grafts feel stable and readable in play.

## Candidate grafts
- **Distance -> Path**
  - Make a route feel longer or shorter.
- **Binding -> Zone**
  - A local area behaves like it holds motion or pulls inward.
- **Moon -> Ground**
  - A night-like local law distinct from Sun.
- **Ground -> Sky**
  - Invert the current sky-to-ground idea into a heavier, grounded patch.
- **Beginning -> Door/Container**
  - An interaction snaps back to an earlier state.

## Guardrails
- Add one graft at a time.
- Prefer zone, path, or anchor-based behavior over global rules.
- Reuse existing cooldowns, active limits, particles, and registry patterns.
- Avoid new freeform combination systems.

## Done when
- Each added graft has a clear one-line fantasy.
- The behavior is obvious within a few seconds of use.
- It can be explained briefly in README/help.
